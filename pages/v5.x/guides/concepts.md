# Core Concepts

Webpack builds a dependency graph and emits optimized assets.

## Entry

The starting point for dependency graph traversal.

## Output

Where webpack writes bundles and assets.

## Loaders

Transform non-JavaScript assets into modules webpack can process.

## Plugins

Tap into compilation hooks to extend behavior.

## Modes

- `development`: fast rebuilds, readable output
- `production`: optimized output for deployment
